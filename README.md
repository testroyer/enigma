# Enigma

A C/C++ implementation of the historic Enigma machine, used for encryption and decryption.

## Features

- Simulates the original Enigma machine logic
- Configurable rotors, reflector, and plugboard
- Command-line interface for encryption/decryption

## Getting Started

### Prerequisites

- C or C++ compiler (e.g., GCC, Clang)
- Make (optional)

### Build

```bash
git clone https://github.com/yourusername/enigma.git
cd enigma
make
```

### Usage

```bash
./enigma --help
```

## Project Structure

- `src/` — Source code
- `include/` — Header files
- `tests/` — Unit tests
